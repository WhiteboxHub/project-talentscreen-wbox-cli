🐛 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

Issue: "API Key not found"
  → Solution: export CLAUDE_API_KEY="your-key"

Issue: "Provider 'claude' not recognized"
  → Solution: Run setup_claude_agent.py to verify installation

Issue: Slow responses (expected)
  → Normal: Claude takes 1-3 seconds per form
  → This enables better reasoning

Issue: Form not filled correctly
  → Solution: Enable debug logging and review logs/jobcli.jsonl

Issue: Cannot import claude_agent
  → Solution: Verify jobcli/core/claude_agent.py exists and is readable
